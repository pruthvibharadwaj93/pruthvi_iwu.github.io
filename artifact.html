Introduction

[ Voyager — AI-Powered Trip Planner Voyager is a conversational AI trip planning assistant built with a custom GPT and deployed as a web application via Netlify. ]

Description

[ Designed to make travel planning effortless, Voyager lets users describe their ideal trip — whether a family vacation, business travel, or weekend getaway — and receive personalized itinerary suggestions in real time. The project involved crafting a detailed system instruction prompt to shape the AI's persona and response logic, building a clean chat-based frontend, and integrating it with OpenAI's API. Popular destinations are surfaced as quick-start options, and the interface supports natural, multi-turn conversations. This project demonstrates skills in prompt engineering, API integration, frontend development, and deploying AI-powered applications — combining technical implementation with thoughtful UX design to create a polished, user-ready product. ]

Objective

[ To design and deploy a conversational AI assistant that simplifies travel planning by generating personalized itineraries and destination recommendations through natural, multi-turn dialogue — demonstrating practical applications of prompt engineering, API integration, and full-stack web development. ]

Process

[ Voyager was built by first defining the core use case — a conversational AI travel assistant — and identifying the types of trips and destinations users would most commonly plan. The process centered on prompt engineering: drafting, testing, and iteratively refining a system instruction prompt that shapes the chatbot's persona, scope, and response quality. The model was built on Anthropic's Claude, where the engineered prompt acts as the intelligence layer guiding the assistant's tone and behavior. Evaluation involved testing across diverse travel scenarios — family vacations, business trips, and weekend getaways — using the results to further tune the prompt. Finally, a responsive chat frontend was developed and deployed on Netlify, making Voyager publicly accessible as a fully functional, live web application. ]

Tools & Technologies Used

[ Claude, Netlify, Github, Google Docs, Canva ]

Value Proposition

Unique Value

[ Voyager stands out by combining Claude's conversational intelligence with a carefully engineered system prompt that delivers a focused, travel-specific experience rather than a generic chatbot interaction. Unlike broad AI assistants, Voyager is purpose-built for trip planning — offering curated destination suggestions, context-aware itineraries, and support for diverse travel types all within a clean, intuitive interface. What differentiates it further is the deliberate prompt design that shapes the assistant's persona and keeps responses consistently relevant and actionable. ]

Relevance

[ The project speaks directly to a real-world need — travel planning is time-consuming, fragmented across multiple platforms, and often overwhelming. Voyager consolidates that experience into a single conversational tool, demonstrating how accessible AI deployment can solve everyday problems. It also showcases a practical, replicable framework for building domain-specific AI assistants with minimal infrastructure and no proprietary dataset. ]
